Modals

Modals are dialog boxes that appear on top of the main content to provide critical information or require user interaction. They are used for alerts, confirmations, forms, and other temporary interactions.

Alert Modal

Used to convey important information that the user must acknowledge.

Confirmation Modal

Asks the user to confirm a potentially destructive or irreversible action.

Information Modal

Displays supplementary details or content without requiring an action.

Form Modal

Allows users to input data or complete a task within a modal dialog.
