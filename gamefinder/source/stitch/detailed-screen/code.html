Cosmic Odyssey

Embark on an epic space adventure in Cosmic Odyssey, a thrilling RPG where you explore galaxies, battle alien foes, and uncover ancient secrets.

Available on: PC, PS5, Xbox Series X/S

About the Game

Cosmic Odyssey is a sprawling RPG set in a vast, procedurally generated universe. Players can customize their characters, build their own spaceships, and engage in real-time combat. The game features a rich storyline with multiple branching paths and a dynamic world that reacts to player choices.
